Installation

Requirement

  • Web server with PHP 7.2 running
  • Optionnal : SQL DB server

Put folder content in the web root folder, enjoy

Configuration

To add a new site to your application you need to register your domain. To do it create a folder named like your domain on the Site folder

.env file

If you want to set custom environment configuration or overwrite existing you can write .env file at the root of your site folder like the default .env file :

# APPLICATION PART
APPLICATION_ID=
APPLICATION_NAME=
# APPLICATION CONTEXT
# VAILABLE CONTEXT :
# - Develop
# - Production
APPLICATION_CONTEXT=Development
# SITES PART
# Existing site on application (comma separated)
SITES_DOMAINS=domain.tld
################
### DATABASE ###
################
DATABASE_ENABLE=true
## DATABASE TYPE
## TO USE BY DEFAULT
DATABASE_DRIVER=mysql
################
## MYSQL PART ##
################
# User
MYSQL_USER=db
# Password
MYSQL_PASS=db
# Host
MYSQL_HOST=db
# Port
MYSQL_PORT=3306
# Database name
MYSQL_NAME=db
# LOG PART
# Form of log file name, need to split by date logs
LOG_FORMAT=Ym.d
# Level to log :
# 3 = errors
# 2 = warning
# 1 = notice
# 0 = none
LOG_LEVEL=0
# Log Database query
LOG_QUERY=false
# Template file extention
TEMPLATE_EXT=.html
# Display execution time in header
SHOW_EXECUTION_TIME=true

.routing file

If you want to register routes without tell it into controller you can write a file at the root of your domain folder like this :

{
    "domain.tld": {
        "/": "Site\\Controller\\DefaultController->index",
        "/news/{newstitle}": "Site\\Controller\\DefaultController->news"
    }
}

Hello World

Lets start with a simple hello world rendering

Controller

This is the controller

<?php


namespace Framework\Controller;

use Core\Controller;
use Core\Database\Manager;
use Core\Response;
use Framework\Model\User;

class DefaultController extends Controller
{
    /**
     * Routing define by .routing file at the root of web server (next to index.php)
     * @site domain.tld
     * @route /
     */
    public function index()
    {
        $this->render("index", ["message" => ["hello", "world"]]);
    }
}

Template

The index template location is Site/domain.tld/Resource/index.html and look like :

<p>
    [foreach:message as element]
        '{key:element}:[element}'
    [foreach]
</p>
